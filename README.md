# Portafolio Web - Fabian Mera

Este proyecto es una página web personal y profesional desarrollada en HTML y CSS puro. Aquí encontrarás información sobre mi experiencia, habilidades, proyectos y formas de contacto.

## Estructura

- `index.html`: Página principal con secciones de presentación, sobre mí, portafolio de proyectos y contacto.
- `styles.css`: Hoja de estilos personalizada para un diseño moderno, responsivo y atractivo.

## Características
- Diseño responsivo para dispositivos móviles y escritorio.
- Animaciones y efectos visuales en botones y secciones.
- Sección de portafolio con proyectos destacados.
- Formulario de contacto funcional.
- Cambios de idioma (inglés/español) desde la interfaz.

## Cómo usar
1. Clona este repositorio o descarga los archivos.
2. Abre `index.html` en tu navegador favorito.
3. Personaliza el contenido y los estilos si lo deseas.

## Publicación
Puedes publicar esta web fácilmente en GitHub Pages:
1. Sube los archivos a un repositorio público.
2. Activa GitHub Pages desde la configuración del repositorio.
3. Accede a tu sitio desde el enlace proporcionado por GitHub.
